# ai-coding

A hub repo for small apps and experiments.

## Projects

- `apps/ytmusic-desktop` — YouTube Music desktop wrapper (PySide6)
- `apps/win7-paint-remake` — Windows 7–style Paint remake (Tkinter)
- `apps/scratch-ai-game-builder` — Scratch-style block builder with AI prompt-to-game
- `apps/whiteboardfox-autodraw` — WhiteboardFox browser + auto draw (PyQt6/WebEngine/OpenCV)

## WhiteboardFox AutoDraw quick install

One-command install:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/hoodlandon25/ai-coding/main/scripts/install-whiteboardfox-autodraw.sh)
```

Then run:

```bash
autodraw
```

Manual dependencies (Debian/Ubuntu/Crostini):

```bash
sudo apt-get update
sudo apt-get install -y python3 python3-venv libxcb-cursor0 libxkbcommon-x11-0 libxcb-xinerama0 fontconfig git
```
